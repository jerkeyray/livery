# Bounded canvas

Canvas is local continuous space inside a pinboard region. Use it for mechanisms, plots, annotations, and deliberate overlap; keep macro composition coordinate-free.

Canvas evaluation is declarative and bounded. There is no JavaScript, I/O, recursion, mutation, or unbounded looping. Declare `bleed` when marks extend beyond the local canvas and use clipping or masks explicitly.
