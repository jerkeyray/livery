# Changelog

## 0.1.0-alpha.0

First public alpha of the Livery programmable visual language.

- Adds deterministic single-file visual syntax, validated pinboard layout, bounded local canvases, timelines, SVG/JSON/PNG exports, and technical-editorial standard components.
- Adds retained Web and React runtimes plus `LiveryChatVisual` for streamed agent output.
- Adds machine-readable language intelligence, deterministic repair edits, provider-neutral evaluation, and versioned documentation.
- Keeps legacy flow APIs as deprecated wrappers for this alpha cycle.

The API is pre-1.0. Imports, component packages, additional themes, advanced motion, chart components, and geometric morphing remain unstable or experimental. Remote image access is disabled unless an HTTPS host is explicitly allowed.
